# Argument Audit (论点审计)

Evaluate the logical structure, evidence quality, and persuasiveness of arguments — detect fallacies, rate strength, and suggest concrete improvements.

## What it does
- Map argument structure: identify main claim, premises, reasoning chains, and missing links
- Detect logical fallacies: ad hominem, straw man, false dilemma, slippery slope, circular reasoning, and more
- Rate argument strength across 5 dimensions (clarity, validity, soundness, completeness, fairness) with actionable recommendations

## Example scenarios

**1. 分析我跟同事在微信上的吵架记录**
> 👤 我跟同事在微信群里因为一个需求吵起来了，他说的我觉得不对但是又不知道怎么反驳，帮我分析一下。
> 🤖 导出微信聊天记录，进行论证分析——1）识别双方的论点、论据和结论；2）发现逻辑谬误（如人身攻击、滑坡谬误、虚假两难）；3）评估证据强度。给出结构化反驳建议：先承认对方合理部分，再用事实/数据支持自己的立场，最后建议私下沟通而非群内争论。

**2. Analyze an article's argument**
> 👤 帮我分析这篇文章的论证逻辑。
> 🤖 Argument structure diagram identifying main claim and supporting premises, evidence quality assessment with credibility/relevance/sufficiency ratings, logical fallacy detection, and overall strength score (1-10).

**3. Prepare for a debate**
> 👤 找出我论点中的薄弱环节，我要准备辩论。
> 🤖 Vulnerability scan identifying unsupported assumptions, missing evidence, potential counterarguments, and 3-5 concrete improvement suggestions to strengthen the position.
